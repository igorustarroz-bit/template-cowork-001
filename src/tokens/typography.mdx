# Typography

## Intro

The Euro6000 Design System uses a single font family, **Geist**, applied
through 28 named text styles (`Title`, `Body`, `Labels`, `CTA`,
`CTA-Link-Footer`, `Forms`). Every style is **responsive**: font size and
line height are bound to Figma's `Responsive` variable collection and vary
per breakpoint, while font family, weight and letter-spacing come from the
`Tipography/*` primitives and stay fixed. Each style has a corresponding
CSS class `.type-<slug>` (defined in `src/index.css`) that applies the
correct value at every breakpoint via media queries.

Do not edit these values by hand — regenerate from Figma if a text style changes.

## Tokens

| Group | Styles | Weight | Letter-spacing | Notes |
| --- | --- | --- | --- | --- |
| `Title/01`–`07` | 7 | 700–900 | −0.8px | Headline scale |
| `Body/01`–`06` | 6 | 400 | 0px | Body copy |
| `Labels/01`–`02` | 2 | 400 | 0.5px | Small caption-like text |
| `CTA/01`–`03` | 3 | 500–600 | 0px | Button/link labels |
| `CTA-Link-Footer/01`–`02` | 2 | 400 | 0px | Footer link styles |
| `Forms/*` | 8 | 400 | 0px | Input text, label-in/out, info messages, big (M) and small (S) sizes |

Each style is bound to two Figma variables — e.g. `Title/07` →
`Typography/Title 7/SZ` (font-size) and `Typography/Title 7/LH` (line-height) —
both resolved per breakpoint from the `Responsive` collection. See
`allTextStyles` in `src/tokens/typography.ts` for the full, flat list with
per-breakpoint values.

Breakpoints: `xs` 375 · `sm` 480 · `m` 768 · `lg` 1024 · `xl` 1440 ·
`xxl` 1620 · `xxxl` 1920 (px) — same scale as `Tokens/Spacing and breakpoints`.
